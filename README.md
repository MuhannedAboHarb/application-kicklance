# 📊 MyDashboard Project

واجهة Dashboard بسيطة باستخدام **HTML & CSS** لعرض بيانات النظام والمبيعات بشكل منظم وجذاب.

---

## 🚀 Features

* ✅ تصميم Dashboard بسيط ومرتب
* ✅ Navbar يحتوي على روابط تنقل
* ✅ كروت (Cards) لعرض معلومات سريعة
* ✅ جدول لعرض بيانات النظام
* ✅ أزرار بحالات مختلفة (نشط - قيد المراجعة - متوقف)
* ✅ تصميم متجاوب جزئيًا (Responsive Layout باستخدام Flexbox)

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox

## 🎨 UI Sections

### 🔹 Header

* يحتوي على:

  * اسم النظام (MyDashboard)
  * روابط التنقل
  * اسم المستخدم

---

### 🔹 Cards Section

* 5 كروت لعرض:

  * المبيعات
  * التحليل
  * التواصل
* كل كارد يحتوي:

  * صورة
  * عنوان
  * وصف
  * زر

---

### 🔹 Table Section

* جدول يعرض:

  * الأقسام
  * عدد المستخدمين
  * الحالة
  * آخر تحديث
